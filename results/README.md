# Results

`final_paper_tables/` contains final metric tables from the frozen v0.2k scorer package.

`scored_runs/` contains per-run raw outputs, item scores, triad scores, aggregate metrics, slice metrics, and scorer manifests for the official and diagnostic runs. Local model-cache paths in public metadata have been sanitized to model repository identifiers; model revisions and scoring outputs are preserved.

`lw3_analysis/` contains derived analysis tables added during the LW3 paper revision, including Wilson intervals, role-level diagnostics, warrant-source metrics, and shortcut baselines.
