# Data

`frozen_release/` contains the frozen 1,500-item LangWarrant benchmark release. The frozen item identifiers preserve the legacy `SLD_v1_0` prefix for checksum and provenance compatibility.

`inventory/` contains distribution and run-inventory tables used by the paper.

No real customer records, production account credentials, or private access tokens are intentionally included in this release. The workflow records are constructed benchmark instances.
